¿Que es CSS Grid Layout?

Se puede utilizar para lograr muchos diseños diferentes. También se destaca por permitir dividir una página en áreas o regiones principales, por definir la relación en términos de tamaño, posición y capas entre partes de un control construido a partir de HTML primitivo.

• header - Define un encabezado de un documento o una sección

• nav - Define un conjunto de enlaces para navegación o menú

• section - Define una sección en un documento

• article - Usado para definir un contenido independiente

• aside - Usado para definir un contenido lateral o barra lateral

• footer - usado para definir un pie de página de un documento o sección

¿Cual es su funcion CSS Grid Layout?

Permite un autor alinear elementos en columnas y filas. Sin embargo, con CSS grid son posibles muchos más diseños y de forma más sencilla que con las tablas. Por ejemplo, los elementos secundarios de un contenedor de cuadrícula podrían posicionarse para que se solapen y se superpongan, de forma similar a los elementos posicionados en CSS.

CSS Grid también permite utilizar una grilla con mucha versatilidad utilizando la propiedad grid-template-areas al contenedor principal, esto permite usar una plantilla en la que asignamos grid-area a cada contenedor dentro de la grilla, en la sección de HTML se observa que se ha agregado un "div" con la clase “contenedor-principal”, en la sección de CSS se le asigna a la propiedad gid-template-areas una plantilla de la grilla donde el nombre de cada área representa el espacio que ocupará en la pantalla
